# 仿掌上英雄联盟云顶之弈 - 微信小程序版

包含英雄、英雄详情、装备、阵容、阵容详情、棋子概率 6 个页面

## 功能
1. 英雄列表切换、排序、筛选、搜索、详情跳转
2. 英雄详情、跳转
3. 装备列表、搜索、筛选
4. 阵容列表、上拉加载更多
5. 云函数
6. 优化页面提示，下拉刷新，英雄页渲染优化
7. url提取（wxs）
8. 阵容详情页（使用本地存储）
9. 阵容详情添加分享功能
10. 棋子概率页
11. 版本更新监听

## 数据库数据
可以通过云数据库的导入功能进行导入。这里数据是拿的官网数据，但是阵容部分数据有缺失没有进行补充。
